# Task 1: Data Cleaning and Preprocessing

## Objective

The objective of this task is to clean and preprocess the Netflix dataset to improve data quality and prepare it for further analysis.

---

## Dataset

Dataset: Netflix Titles Dataset

The dataset contains information about Netflix Movies and TV Shows, including title, director, cast, country, release year, rating, duration, genre, and description.

---

## Tools Used

### Microsoft Excel

* Data inspection
* Filtering and sorting
* Missing value identification
* Duplicate detection

### Python

* Pandas
* NumPy

---

## Data Cleaning Steps Performed

### 1. Dataset Inspection

* Examined dataset structure
* Reviewed columns and data types
* Identified missing values and duplicates

### 2. Duplicate Removal

* Removed duplicate records to ensure data consistency

### 3. Missing Value Handling

* Identified null values
* Standardized missing entries
* Treated missing values appropriately

### 4. Text Cleaning

* Removed unnecessary spaces
* Standardized text formatting
* Improved categorical value consistency

### 5. Column Standardization

* Converted column names to lowercase
* Replaced spaces with underscores

### 6. Date Conversion

* Converted date fields into datetime format

### 7. Feature Engineering

Created additional columns:

* year_added
* month_added
* content_age
* primary_genre
* duration_value
